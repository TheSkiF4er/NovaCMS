# Security Policy

## Supported Versions
NovaCMS follows semantic versioning. Security fixes are provided for:
- The latest released minor version of the current major line.

## Reporting a Vulnerability
Please report security vulnerabilities privately to:

- **security@cajeer.com**

Include, when possible:
- Product version(s) and environment details
- Steps to reproduce / proof of concept
- Impact analysis
- Any suggested mitigation

We aim to acknowledge valid reports within 72 hours.
