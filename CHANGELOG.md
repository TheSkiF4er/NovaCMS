# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project adheres to Semantic Versioning.

## [1.0.0] - 2025-12-12

### Added
- Monorepo baseline (apps + packages) with Turborepo and pnpm workspaces.
- API service scaffold (NestJS) with REST health endpoint and GraphQL placeholder.
- Admin panel scaffold (Next.js) with basic layout and routing.
- Prisma database layer package with schema baseline and migration tooling.
- Plugin SDK interfaces and example plugin skeleton.
- TypeScript SDK package (REST/GraphQL client placeholders).
- Shared UI package and minimal design system primitives.
- Docker Compose for local Postgres, Redis, and S3-compatible object storage (MinIO).
- CI workflow (lint, typecheck, test, build), conventional commits and release notes templates.
- Repository policies: Code of Conduct, Contributing, Security policy, Apache-2.0 LICENSE, NOTICE.
