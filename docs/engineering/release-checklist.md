# Release checklist

- [ ] Ensure `LICENSE` and `NOTICE` are correct
- [ ] Bump versions (root + packages) and update `CHANGELOG.md`
- [ ] Tag release: `vX.Y.Z`
- [ ] Create GitHub Release notes from `docs/releases/X.Y.Z.md`
- [ ] Verify CI is green
- [ ] Generate SBOM (optional)
- [ ] Publish packages (optional)
