# Release 1.0.0 (2025-12-12)

This is the initial public release of NovaCMS source distribution.

## Highlights
- Monorepo baseline with pnpm workspaces and Turborepo.
- API scaffold (NestJS) and Admin scaffold (Next.js).
- Database layer with Prisma schema and seed data.
- Plugin SDK and official SDK placeholders.

## Upgrade notes
N/A (first release).

## Known limitations
- This release is a scaffold intended to establish contracts and repository hygiene.
- Auth, RBAC, content entry storage, and visual builder are intentionally minimal.
